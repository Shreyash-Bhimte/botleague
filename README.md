# ⚡ BotLeague — India's Ultimate Robotics Arena

A frontend web application for **BotLeague**, India's national ecosystem for competitive robotics. Built as part of a Full Stack Developer assignment for Botmakers Pvt Ltd.

---

##  Tech Stack

- **React 18** (with Vite)
- **Plain CSS** (component-scoped, no frameworks)
- **JavaScript ES6+**

---

##  Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/botleague.git
cd botleague

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
